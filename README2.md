# URL Cleaner

A Python tool that removes tracking parameters from URLs.

## Description
Many websites add tracking parameters like `utm_source` and `utm_medium` to URLs.
This project removes those parameters and returns a cleaner version of the link.

## Features
- Removes tracking parameters
- Keeps important query parameters
- Simple command-line tool
- Uses Python standard library

## Technologies Used
- Python
- urllib.parse module

## How to Run
1. Clone the repository
2. Navigate to the project folder
3. Run:

   python url_cleaner.py

4. Enter a URL when prompted
